# Mon vieux Grimoire
## Description
Ce projet est une application web de critique de livres. Il permet aux utilisateurs de consulter des critiques de livres, d'en ajouter de nouvelles, et de les modifier ou supprimer.
## Comment lancer le projet ?
### Avec npm

Faites la commande `npm install` pour installer les dépendances puis `npm start` pour lancer le projet.
Le projet a été testé sur node 19.
